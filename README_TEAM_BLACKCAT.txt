# TEAM BLACKCAT (Addon for CYBER FLAY v9)
- Tambahan fitur: Phone Lookup PLUS (Deep 50+ sites)
- Cara pakai (Termux/Linux):
  ```bash
  unzip team_blackcat.zip
  chmod +x team_blackcat.sh
  ./team_blackcat.sh
  ```
- Di menu Phone Lookup (fitur asli), akan muncul pertanyaan opsional:
  `Jalankan DEEP SCAN 50+ situs juga? (y/N)`
  Jawab `y` untuk menjalankan pemindaian tambahan tanpa mengubah alur asli.
